# Superstore Sales Dashboard (Excel)

## Project Overview

This project is an interactive sales dashboard built in Microsoft Excel using the Sample Superstore dataset from Kaggle.

The dashboard helps analyze sales performance using Pivot Tables, Pivot Charts, KPIs, and Slicers.

---

## Dashboard Features

- Total Sales KPI
- Total Customers KPI
- Average Delivery Days
- Sales by Month
- Sales by Sub-Category
- Orders by Region
- Order Percentage by Segment
- Order Percentage by Ship Mode
- Top 10 States by Orders
